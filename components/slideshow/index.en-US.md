## 何时使用

滑动展示内容，上滑收起内容

## API

| 参数 | 说明 | 类型 | 默认值                         |
| --- | --- | --- | ------------------------------ |
| open | 是否展示内容 | boolean | false |
| title | panel ｜ string \| React.Node | - |
| onChange | 动态切换展示内容 | () => voide | - |
| onEnd | 动画执行完回调 | (exists: boolean) => {} | - |
| duration | 下拉上滑动画执行时间 | number | 500 |
